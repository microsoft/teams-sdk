# AI Integrations

The Teams SDK for Python is intentionally focused on the Teams platform primitives — activity handling, conversations, cards, proactive messaging, and bot lifecycle. AI capabilities are built by composing the SDK with best-in-class external libraries.

This section shows two integration patterns built on top of `microsoft-teams-apps`:

- **[MCP Server (FastMCP)](./mcp-server)** — expose your Teams bot as a [Model Context Protocol](https://modelcontextprotocol.io/introduction) server using the official [`mcp`](https://github.com/modelcontextprotocol/python-sdk) Python SDK. Lets external AI agents notify Teams users, ask them questions, and request approvals.
- **[Agent Framework](./agent-framework)** — build an AI agent powered by [Microsoft Agent Framework](https://github.com/microsoft/agent-framework) and Azure OpenAI, with streaming responses, MCP tools, citations, and per-conversation memory.

Both patterns are based on the runnable samples in the [`microsoft/teams.py` examples](https://github.com/microsoft/teams.py/tree/main/examples).
