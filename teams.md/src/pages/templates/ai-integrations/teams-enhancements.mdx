# Enhance the Teams Experience
This layer defines how agent output is enriched into a Teams-native experience - adding structure, interactivity, and metadata on top of the generated text.

## Streaming 

Streaming lets the agent send partial responses back to Teams as they are generated, instead of waiting for the full completion.
Each text chunk is emitted immediately into the chat.
```python
@app.on_message
async def handle_message(ctx: ActivityContext[MessageActivity]):
    async for chunk in agent.run(ctx.activity.text or "", stream=True):
        if chunk.text:
            ctx.stream.emit(chunk.text)
```


## Suggested prompts

Suggested prompts give the user one-click follow-up questions after a reply.
In Teams they render as chips under the message; tapping one sends the `value` back as a normal user message, so the same `on_message` handler picks it up — no extra routing required.

Define prompts using `CardAction` and attach them to the reply via `with_suggested_actions`:

```python
from microsoft_teams.api import CardAction, CardActionType, SuggestedActions

_SUGGESTED_PROMPTS = [
    CardAction(
        type=CardActionType.IM_BACK,
        title="How do I stream in teams.py?",
        value="How do I stream in teams.py?",
    ),
    CardAction(
        type=CardActionType.IM_BACK,
        title="How do I create an Adaptive Card in teams.py?",
        value="How do I create an Adaptive Card in teams.py?",
    ),
]

reply= MessageActivityInput().with_suggested_actions(
    SuggestedActions(to=[ctx.activity.from_.id], actions=_SUGGESTED_PROMPTS)
)
```

The `to` field scopes the chips to a specific user — important in group chats where only the asker should see the follow-ups.

## Marking AI-generated content and enabling feedback
`add_ai_generated()` marks the message as system-generated, ensuring it is clearly labeled as AI output within Teams.
`add_feedback(mode="custom")` enables built-in thumbs up/down controls and allows you to surface a custom feedback flow when users respond.

```python
from microsoft_teams.api import MessageActivityInput

reply = (
    .add_ai_generated()
    .add_feedback(mode="custom")
)
```
See [Feedback](../../in-depth-guides/feedback) for the full form-handling story.

## Citations

In the message handler, filter citations to those referenced in the final text:

```python
from microsoft_teams.api import CitationAppearance, MessageActivityInput

def attach_citations(reply, full_text, citations):
    used = extract_referenced_ids(full_text)

    for c in citations:
        if c.position in used:
            reply.add_citation(
                position=c.position,
                appearance=CitationAppearance(
                    name=c.title,
                    abstract=c.description,
                    url=c.url),
                )
```
