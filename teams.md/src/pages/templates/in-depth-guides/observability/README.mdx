# Observability

Understanding what your agent is doing at runtime is critical for debugging and production monitoring. The Teams SDK provides built-in support for structured logging and middleware-based tracing so you can inspect activity flow, catch errors, and measure performance without third-party dependencies.

## What's in this section

- [Logging](logging) — configure log levels, custom loggers, and structured output
- [Middleware](middleware) — intercept and inspect activities at every stage of the pipeline
