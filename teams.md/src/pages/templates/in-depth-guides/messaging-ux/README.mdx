# 📖 Messaging UX

Messaging UX features let your agent enrich conversations beyond plain text replies. These capabilities give your agent low-friction ways to acknowledge messages, signal status, and respond to user interactions directly within the Teams message surface.

- [Message Reactions](./message-reactions.mdx) — add, remove, and receive emoji reactions on messages.
