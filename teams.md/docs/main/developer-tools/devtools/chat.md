# 💬 Devtools chat

:::warning Deprecated
DevTools is deprecated and will be removed in a later version. Use the [Microsoft 365 Agents Playground](../agents-playground) for local testing of your agent. For details, see the [Microsoft Learn guide](https://learn.microsoft.com/en-us/microsoft-365/agents-sdk/test-with-toolkit-project).
:::

The DevTools chat page was a Teams-like chat window that let you interact with your agent locally. It supported sending messages, attachments, reactions, edits, deletions, streaming responses, and feedback without requiring an app id or sideloading.

For local testing going forward, install the [Microsoft 365 Agents Playground](../agents-playground).
