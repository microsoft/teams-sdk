# 🔍 Inspect activities

:::warning Deprecated
DevTools is deprecated and will be removed in a later version. Use the [Microsoft 365 Agents Playground](../agents-playground) for local testing of your agent. For details, see the [Microsoft Learn guide](https://learn.microsoft.com/en-us/microsoft-365/agents-sdk/test-with-toolkit-project).
:::

The DevTools Activities page logged every incoming and outgoing activity (messages, reactions, invokes, and more) and let you inspect the JSON payload of each. For local testing going forward, use the [Microsoft 365 Agents Playground](../agents-playground), which exposes the same wire traffic via its Log Panel.
